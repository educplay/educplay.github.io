Educplay v2
Instrucciones:
- Abre index.html en tu navegador.
- Regístrate en 'Registro / Login' para que tus puntos se guarden.
- Juega: contar globos, adivina la letra, memoria.
- Lee: 3 libros con botón 'Marcar como leído' (+10 puntos).
- Escucha: 3 canciones (coloca archivos mp3 dentro de la carpeta 'audio' con nombres abc.mp3, numbers.mp3, colors.mp3) y usa 'Marcar como escuchada'.

Los datos se guardan en localStorage del navegador (no se necesita servidor).
